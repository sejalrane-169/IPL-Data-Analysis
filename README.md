# 🏏 IPL Data Analysis Dashboard 2008-2025

## Overview
End-to-end data analysis project on 18 seasons of IPL cricket 
covering 1,158 matches and 1,34,190 ball-by-ball deliveries.

## 🔗 Live Tableau Dashboard
[Click here to view interactive dashboard](https://public.tableau.com/app/profile/sejal.rane7808/viz/IPL-Data-Analysis-Dashboard-2008-2025/Dashboard1)

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Microsoft Excel (Pivot Tables, Dashboard)
- Tableau Public (Interactive Dashboard)

## Dataset
- Source: Kaggle — IPL Complete Dataset 2008-2025
- matches.csv — 1,158 rows, 25 columns
- deliveries.csv — 1,34,190 rows, 18 columns

## Key Insights
- Royal Challengers Bangalore leads all-time wins with 145+ victories despite never winning the IPL title
- 53% of toss winners chose to field first across all seasons
- Average first innings score rose from 168 in 2008 to 183 in 2025
- Mumbai Indians have won the most IPL titles

## Project Structure
- IPL_Analysis.ipynb — Python data cleaning and analysis
- IPL_Excel_Dashboard.xlsx — Excel pivot table dashboard
- PNG files — Chart screenshots from Python analysis

## How to Run
1. Download matches.csv and deliveries.csv from Kaggle
2. Open IPL_Analysis.ipynb in Jupyter Notebook
3. Run all cells in order
